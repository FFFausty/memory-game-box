# Netlify 快速部署指南 - 经典游戏盒

## 🚀 立即开始部署

### 步骤1：访问Netlify
1. 打开浏览器访问：**https://app.netlify.com/**
2. 点击 "Sign up" 注册账号（或登录）

### 步骤2：连接GitHub
1. 选择 "GitHub" 作为登录方式
2. 授权Netlify访问你的GitHub账号

### 步骤3：选择仓库
1. 点击 "New site from Git"
2. 选择 "GitHub"
3. 搜索并选择：**FFFausty/memory-game-box**

### 步骤4：配置部署设置
在部署配置页面设置：
- **构建命令**: `npm run build`
- **发布目录**: `build/web-mobile`
- **Node版本**: 18（自动检测）

### 步骤5：开始部署
1. 点击 "Deploy site"
2. 等待构建完成（约3-5分钟）
3. 获取部署链接

## 🌐 部署后访问

部署完成后，你会获得一个类似这样的链接：
**https://amazing-memory-game-box-123456.netlify.app**

## 🔧 自定义域名（可选）

### 添加自定义域名：
1. 在Netlify控制台选择你的站点
2. 点击 "Domain settings"
3. 添加自定义域名
4. 配置DNS记录

## 📊 部署状态监控

部署过程中可以查看：
- ✅ 构建日志
- ✅ 部署状态
- ✅ 错误信息（如果有）

## 🆘 常见问题解决

### 构建失败怎么办？
1. 检查构建日志中的错误信息
2. 确认Node版本为18
3. 检查依赖是否正确安装

### 白屏问题？
1. 检查浏览器控制台错误
2. 确认资源路径正确
3. 检查重定向配置

- ✅ 所有资源加载正常
